HTML Canvas "Space Invaders"

Key skills: JavaScript, HTML Canvas

Summary

This was a fun little project to work with the HTML canvas element more, in particular to work on animations and user interaction. A space invaders inspired vertical scrolling shooter of the like from my youth seemed like a fun way to go that!

Controls:

  • M - Mute music
  • P - Pause
  • Arrow keys - Move
  • Space - Fire!
