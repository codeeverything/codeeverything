HTML Canvas IDE

Key skills: JavaScript, HTML Canvas

Summary

Inspired by Mozilla's Bespin project I wanted to try and build an IDE within the browser using the Canvas element.

Since the HTML Canvas element is a drawing surface, i.e. an image, this brought with it a list of interesting things to think about including editing of text, rendering of text, syntax highlighting of code, coords -> character position mapping and selection handling.

Please note: This is an old project and is far from complete. I have tested in Chrome 38 and it seems stable, but it is not assured.
