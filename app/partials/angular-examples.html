Password

A reusable password input with customisation options. This is limited to one instance per page as it relies on an element with an ng-form attribute and hardcoded name, along with a hardcoded name for the password input. In reality you would probably include this in a form with a name anyway and only have one password input (a "Confirm password" input would be subject to different rules, i.e. is it equal to the password entered, and therefore a different directive).
