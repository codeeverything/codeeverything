CodeEverything.com

Key skills: Angular JS, MVC, SPA, HTML, CSS

Summary

CodeEverything.com is my place of the web, but also an opportunity for me to work with new technologies.

As such I've designed CodeEverything as a Single Page Application (SPA), using Angular JS as my front-end framework of choice.

In Detail

CodeEverything is a simple SPA, in fact it's probably a bit of a stretch to call it an "application" at all (at the time of writing at least). I'm using Angular JS, in this specific context, for the routing it provides.
